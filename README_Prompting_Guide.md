**AI 101 Training Series | LachesisAI**
**Content current as of: March 23, 2026 | Next review: March 2027**

---

## Overview

A consolidated, practitioner-ready training guide for Microsoft 365 Copilot prompt engineering. This document replaces the two-part prompting series from the original AI 101 curriculum and reflects the current Microsoft-official GCES (Goal, Context, Expectations, Source) four-element framework.

Produced as a portfolio artifact for Texas DIR Req 5944 (IT Course Writer / Training Developer) and as a reusable LachesisAI training deliverable for enterprise Copilot adoption programs.

---

## What's Inside

| Section | Content |
|---|---|
| 1. What is Microsoft 365 Copilot? | Platform overview, licensing, underlying model (GPT-4o / GPT-5.1 agents) |
| 2. Core Capabilities by Application | Q1 2026 feature table: Word, Excel, PowerPoint, Outlook, Teams, Copilot Chat, Loop, Copilot Studio |
| 3. Four-Element Prompt Formula | GCES framework with comparison to RTF and GRASP+Q |
| 4. Before/After Prompt Examples | Email, data analysis, and SOP prompts — weak vs. strong |
| 5. Iterative Refinement | Feedback loop workflow; Prompt Gallery documentation |
| 6. Copilot-Specific Syntax | /slash commands, @mentions, document grounding, 128K token input |
| 7. Best Practices and Pitfalls | Do/avoid table; prohibited data categories |
| 8. Resources | MS-4005, Copilot Lab, Prompt Gallery, Microsoft Learn links |
